Details about the crosswalks included in this folder:



2010_ZCTA_2021_Census_Places_Crosswalk.csv
	How this was created: https://urbanorg.app.box.com/file/1013082171225
	Variables of interest:
		ZCTA5CE10 = 2010 ZCTA code (same as GEOID10)
		PLACEFP = 2021 Census Place designation code
		ZCTAinPlace = the portion of the ZCTA that falls into that Census Place (use for weighting)
